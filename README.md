# Web-Based Lexical Analyzer

A web application that converts source code into lexemes and tokens with maximum accuracy.

## Features
- Detailed token categorization (18+ operator types)
- Accurate line and column number tracking
- File upload support (.txt, .py, .cpp, .docx)
- Error handling (never crashes)
